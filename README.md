# Credit Scoring Project

## Описание

Проект посвящён задаче кредитного скоринга - предсказанию вероятности дефолта клиента на основе его финансовых характеристик. Используется датасет Give Me Some Credit (Kaggle). Модель решает задачу бинарной классификации: 0 - клиент надёжен, 1 - возможен дефолт.

## Установка

Установите зависимости командой:
pip install -r requirements.txt

## Данные

Скачайте датасет с Kaggle и поместите файл в папку:
data/cs-training.csv

## Запуск

Для запуска полного пайплайна выполните:
python src/pipeline.py

## Метрики

В проекте используются основные метрики бинарной классификации: ROC-AUC, Precision, Recall и F1-score. Основной упор сделан на F1-score из-за сильного дисбаланса классов.

## Особенности реализации

Проект реализован в виде модульного пайплайна и включает: обработку пропусков, масштабирование признаков, работу с дисбалансом классов, подбор оптимального порога и разделение логики по модулям.

## Дальнейшее развитие

Возможные направления улучшения проекта: подбор гиперпараметров, использование более сложных моделей, расширенный feature engineering и последующий деплой модели.
